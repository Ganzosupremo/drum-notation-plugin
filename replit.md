# Drum Notation Plugin for Obsidian

## Project Overview

This is an **Obsidian plugin** that renders drum notation from fenced `drums` code blocks into SVG graphics inside Obsidian notes. It is focused on groove charts (not classical engraving) and is designed for worship, modern session charts, and Obsidian-based workflows.

**This is not a web application.** It is a TypeScript library bundled for use inside the Obsidian desktop app.

## Tech Stack

- **Language**: TypeScript
- **Bundler**: esbuild
- **Package Manager**: npm
- **Target**: Obsidian plugin API (CommonJS bundle)

## Project Structure

```
src/
  main.ts          — Plugin entry point; registers the code block processor
  parser.ts        — Parses drum notation text into data structures
  settings.ts      — Plugin settings management
  types.ts         — Shared TypeScript interfaces
  notation/
    layout/        — Beat layout, beam grouping logic
  renderer/
    renderNotation.ts  — Orchestrates SVG rendering
    notes/         — Per-instrument renderers (HH, SD, BD)
    svgHelper.ts   — SVG utility helpers
main.js            — Bundled output (generated by build)
manifest.json      — Obsidian plugin manifest
styles.css         — CSS for rendered notation
```

## Build Commands

```bash
npm run dev       # Watch build with inline sourcemaps (development)
npm run build     # Type-check + minified production build
npm run lint      # ESLint
```

## Manual Install into Obsidian

Copy these files into `<Vault>/.obsidian/plugins/drum-notation-plugin/`:
- `main.js`
- `manifest.json`
- `styles.css`
- `Bravura.woff2` — SMuFL music font (required for notehead glyphs)

## User Preferences

- Maintain the existing modular src/ structure.
